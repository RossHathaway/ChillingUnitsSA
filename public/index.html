Almond Climate Comparison

The purpose of this website is to provide climate data from Dark Sky comparing the climates of almond-producing regions in Califorina and Southern Africa. California is the largest producer of almonds in the world, and the climate is ideal (source). I have wondered for some time if almonds could be grown in Southern Africa, so I wanted to collect climate data to compare the two regions that could be analyzed to determine the suitability of Southern Africa's climate for almond growing.

The data is a dump file from a MySQL database, and this file can be used to reproduce the database. Directions are here. The dump file will create a new database on the user's computer when reloaded.
